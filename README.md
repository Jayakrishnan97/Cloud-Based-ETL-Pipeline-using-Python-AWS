# Cloud-Based-ETL-Pipeline-using-Python-AWS
Built an end-to-end ETL pipeline using Python to transform raw data and load optimized Parquet datasets into AWS S3, enabling analytics via AWS Athena. Applied cloud best practices including IAM-based access, partitioned storage, and efficient data formats.
## Description
This project cleans and analyzes Airbnb dataset using Python and Pandas.

## Tech Stack
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Steps
1. Data cleaning
2. Handling missing values
3. Feature engineering
4. Export to Parquet
